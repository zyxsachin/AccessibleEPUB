Vorbemerkungen</h2><h3 id="toc_10">0.1. Mengen</h3><p>Eine <em>Menge</em> ist nach Cantor eine Zusammenfassung $$M$$ von bestimmten wohlunterschiedenen Objekten unserer Anschauung oder unseres Denkens (welche Elemente von $$M$$ genannt werden) zu einem Ganzen.</p><p></p><p></p><p></p><p><strong>Notation</strong>: geschweifte Klammern $$\left\{ \right\}$$.</p><p></p><p></p><p></p><h4 id="toc_11">Beispiel 0.1. Notationen:</h4><ul><li><font size="1">$$M = \{ 1,2,3 \}$$</font></li><li><font size="1">$$M = \{ x: x \text{ ist Vielfaches von }7 \}$$ oder $$\{ x \in N: x \text{ Vielfaches von }7 \}$$</font></li></ul><p>Weitere Grundnotation: Doppelpunkt zur Kennzeichnung von <em>Definitionen</em>.</p><h4 id="toc_12">Beispiel 0.2.</h4><p>Wollen die Funktion $$f$$ definieren. Schreibe (z.B.) $$f(x):=x^2$$. Nur bei einer Neudefinition, nicht bei einer Gleichung. Oder $$a:=15, f \text{ heißt injektiv } : \Leftrightarrow \text{ Für alle } a, \tilde{a} \in M \text{ mit } a \neq \tilde{a}$$ gilt...</p><p></p><p></p><p>$$a \in M$$ (oder $$M \ni a$$): $$a$$ ist Element von $$M$$; $$M$$ enthält $$a$$.<br/>$$a \notin M$$ (oder $$M \notni a$$): analog s.o.<br/>$$M = N$$: $$M$$ enthält die selben Elemente wie $$N$$<br/>$$M \neq N$$: analog s.o.<br/>$$M \subset N$$ (oder $$M \subseteq N$$): $$M$$ ist Tielmenge von $$N$$, d.h. jedes Element von $$M$$ ist auch ein Element von $$N$$; Gleichheit der Mengen ist erlaubt.<br/>$$N \supset M$$ (oder $$N \supseteq M$$): $$N$$ ist Obermenge von $$M$$; analog<br/>$$M \subsetneqq N$$: $$M$$ ist echte Teilmenge von $$N$$; $$M \neq N$$<br/>$$\emptyset$$: leere Menge</p><p></p><p></p><p>$$M \cup N = \{ a: a \in M \text{ oder } a \in N\}$$ (Vereinigungsmenge)<br/>$$M \cap N = \{ a: a \in M \text{ und } a \in N\}$$ (Schnittmenge)<br/>$$M \setminus N = \{ a: a \in M \text{ und } a \notin N\}$$ (Komplementmenge)</p><p></p><p></p><p>$$M,N$$ heißen disjunkt, wenn $$M \cap N = \emptyset$$<br/>$$P(M) = \{N: N\subset M\}$$: Potenzmenge von $$M$$ (Menge aller Teilmengen)</p><p></p><p></p><p>Beispiel 0.3.</p><p>Beispiel für die Potenzmenge von $$M= \{1,2\}$$:<br/>$$P(M) = \{ \{ 1,2 \} , \{ 1 \} , \{ 2 \} , \emptyset \}$$</p><p></p><p></p><h3 id="toc_13">0.2. Abbildungen</h3><p>Seien $$M,N$$ Mengen. zuordnet.</p><p><strong>Notation:</strong> $$f: M \rightarrow N, a \rightarrow f(a)$$</p><h3>Beispiel 0.4.</h3><p><br/></p><figure><!--RemoveThis--><img title="Beispiel 0.4." class="toRemove" x \rightarrow x^2)$</p><figcaption style="text-align: left;">Eine Abbildung von Real nach Real, macht x zu x quadrat</figcaption></figure><p>$$f_1: M_1 \rightarrow N_1$$ und $$f_2: M_2 \rightarrow N_2$$ heissen <em>gleich</em> (kurz $$f_1 f_2) (identisch), wenn $$M_1=M_2, N_1=N_2$$ und $$f_1(a)=f_2(a)$$ für alle $$a \in M_1$.<br/><br/></p><p>$$f:M \rightarrow N$$ heisst</p><ul><li><em>injektiv</em>, wenn für alle $$a, \tilde{a} \in M$$ mit $$a \neq \tilde{a}$$ gilt: $$f(a) \neq f(\tilde{a})$$; ($$x \rightarrow x$$ ist injektiv, $$x \rightarrow x^2$$ nicht)</li><li><em>surjektiv</em>, wenn für alle $$\tilde{a} \in M$$ ein $$a \in M$$ existiert mit $$f(a)=\tilde{a}$$ (die Bildmenge wird voll ausgeschöpft)</li><li><em>bijektiv</em>, wenn $$f$$ sowohl injektiv als aus surjektiv ist: eindeutige Zuordnung<br/></li></ul><p>Für $$M_1 \subset M$$ heisst $$f(M_1) = f(a): a \in M_1$$ Bildmenge von M_1 (unter $$f$$).<br/>Für $$N_1 \subset N$$ heisst $$f^{-1}(N_1) = a \in M : f(a) \in N_1 <em>Urbildmenge</em> von $$N_1$$ (unter $$f$$).<br/>Sind $$f:M \rightarrow N$$ und $$g: N \rightarrow P$$ Abbildungen, so heisst die Abbildung<br/></p><p></p><p><em>Hintereinanderausführung</em> von $$f$$ und $$g$$.</p><!--EndOfVisibleSection-->
